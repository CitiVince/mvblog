Minimum Viable Blog

This Blog is an attempt to build a Blog from scratch, using {{ framework }} with {{ language }} on {{ platform }} .

The plan is to document my learnings in much detail, especially when something doesn't work. Not being a programmer, the learning curve is bumpy. Sometimes it's due to the lack of a the whole picture, often I miss essential details.

There are a lot of concepts I don't grasp fully yet, and many that I haven't even heard of. Also, don't expect the terminology used here to be correct. If it isn't, please let me know. This blog may be a good read if you have to ask similar questions as I do.

It's less about the outcome and more about the process. This means content as well as appearance may change regularly.

While the domain mvblog.net should be fairly stable (although neither clean nor elegant), beta.mvblog.net is the experimental bit. Once I figured out how to set it up correctly, that is. There is also this github repository that might eventually become useful.

A growing collection of useful resources on the web can be found here: mvblog.net/resources

If you can't bear the mistakes you see, feel free to support my learnings with your insights.
